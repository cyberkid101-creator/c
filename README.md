# Search Proxy Website

A free search proxy website built with HTML, CSS, and JavaScript.

## Features
- Privacy-focused: Routes searches through a proxy to protect user privacy
- Fast and lightweight: Minimal dependencies and quick load times
- Responsive design: Works on desktop, tablet, and mobile devices
- Easy to use: Simple search interface

## How to Use
1. Clone the repository
2. Open index.html in your browser
3. Enter your search query and press Enter or click the search button

## Files
- index.html - Main HTML structure
- styles.css - Styling and layout
- script.js - JavaScript functionality

## Deployment
This site can be easily deployed to GitHub Pages:
1. Go to repository settings
2. Enable GitHub Pages
3. Your site will be available at https://cyberkid101-creator.github.io/c/
